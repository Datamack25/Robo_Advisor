Robo Advisor

Skeleton project for a Markowitz-based robo-advisor with Streamlit UI.

How to view
- Deploy on Streamlit Cloud: point to `app/main.py`
- Or run locally with `streamlit run app/main.py` (if you install requirements)

Structure
- app/: Streamlit app
- core/: optimization, data loaders, performance
- utils/: plotting helpers
- sentiment/: news + sentiment analysis
